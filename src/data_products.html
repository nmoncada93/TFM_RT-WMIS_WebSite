DATA POLICY

  • Users intending to include RT-WMIS data in their publications are requested to notify gAGE/UPC.
  • Commercial use of RT-WMIS data requires prior consultation with gAGE/UPC to establish licensing arrangements.
  • Redistribution of RT-WMIS data without written consent is strictly prohibited.
  • RT-WMIS data may contain errors, and gAGE/UPC disclaims liability for any damages arising from its use.

FORMATS OF PRODUCTS

The main scintillation monitoring indices are:

  • ROTI L1: ROTI for the L1 signal, measured in TECU/min.
  • ROTI GF: ROTI for the Geometry-free combination, measured in TECU/min.
  • S4 L1: Amplitude scintillation index for the L1 signal.
  • σφ L1: Phase scintillation index for the L1 signal, measured in radians.
  • σIF: Phase scintillation index for the Ionosphere-free combination, measured in meters.

Additional data fields and indices include:

- Ndat values indicate the number of measurements used to calculate each index.
- S4 and ROTI calculations include second-frequency measurements (L2 from GPS, L5 from Galileo).
- Column 10 in sphi.tmp denotes the number of prefit measurements used by the Butterworth HPF.

Column No. Field Name
